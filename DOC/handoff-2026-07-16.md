# Aphelion Handoff — 16 July 2026

## สถานะ

งาน auth/admin และ app shell อยู่ใน working tree ของ branch `main` และพร้อมส่งต่อหลังผ่าน build ตรวจสอบแล้ว งานรอบนี้ขยายจาก auth UI ไปสู่ profile/settings/help, admin user management และ roadmap ของ social platform

## งานที่ทำแล้ว

- ปรับ login/register ให้ใช้ auth shell, interactive globe และ password visibility toggle
- เพิ่ม GitHub/Google OAuth flow และ admin OAuth configuration พร้อมการเข้ารหัส provider secret แบบ AES-GCM
- เพิ่ม/ปรับ session, logout, role และ admin unlock flow
- เพิ่ม app header, dashboard navigation, profile editor, settings และ help pages
- เพิ่ม admin shell, user table, user verification API และ admin user routes
- เพิ่ม Prisma `Account` และความสัมพันธ์ที่จำเป็นสำหรับ OAuth
- เพิ่ม dependencies สำหรับ globe/UI และอัปเดต `package-lock.json`
- เพิ่ม `DOC/aphelion-social-platform-roadmap.md` เป็น source of truth สำหรับลำดับงาน social platform

## ตรวจสอบแล้ว / ต้องตรวจต่อ

- รัน `npm.cmd run build` ก่อน merge/deploy
- ตรวจ database migration และ `prisma generate` ให้ตรงกับ environment จริง
- ตั้งค่า `OAUTH_ENCRYPTION_KEY`, provider client ID/secret และ callback URLs ใน deployment
- ทดสอบ OAuth จริงบน browser หลังตั้งค่า provider แล้ว — build สำเร็จอย่างเดียวไม่ยืนยัน OAuth end-to-end
- ตรวจ login, register, logout, profile/settings และ admin actions บน environment ที่มี PostgreSQL/Redis พร้อมใช้งาน
- `Forgot password` ยังไม่มี reset-password flow เต็มรูปแบบ
- ก่อนเปิด social/chat features ให้ทำ email verification, rate limit, password reset และ security gates ตาม roadmap

Callback URLs สำหรับ local:

- `http://localhost:3000/api/auth/oauth/github/callback`
- `http://localhost:3000/api/auth/oauth/google/callback`

## ขั้นตอนส่งต่องานถัดไป
